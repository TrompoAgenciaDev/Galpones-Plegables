


## Trabajo con motion:

## 1.- Cada trabajo hecho con motion, sea crear o modificar uuna sección, tiene que estar totalmente optimizado y no son arreglos o parches para tapar errores. Se tiene que hacer con la mejor performance posible.

## 2.- Si se añade CSS, tiene que adaptarse a todos los navegadores con los webkits necesarios. SIEMPRE.

## 3.- No se modifica el CSS ya existente de un elemento que se modifica excepto que sea algo escencial para la funcionalidad o en caso de que se solicite explícitamente. Es decir, si un elemento tiene fondo azul y se pide editar el elemento, a menos que explícitamente se diga que se cambie el fondo de color, no se cambia. Así con cualquier otro atributo que tenga.
